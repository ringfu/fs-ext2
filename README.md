# ext2 filesystem analysize
我的Linux课程设计： Linux ext2文件系统分析及部分实现

## ext2 结构
+ boot block
+ super block
+ group descriptor block
+ block bitmap
+ inode bitmap
+ inode table
+ data block

## 实现步骤 
1. define ext2 filesystem structure
2. initialize structures
3. functions: ls, cd, mkdir, touch, cat, clear, exit
